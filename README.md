#  ML-Model: Machine Learning пайплайн для торговых сигналов

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

Комплексный ML-пайплайн для получения, обработки и анализа рыночных данных, интеграции с торговой логикой (Pine Script) и доставки результатов через Telegram-бота. Проект создан как исследовательская платформа для тестирования гибридных подходов: классические технические индикаторы + модели машинного обучения.

## Основная концепция

Этот проект служит связующим звеном между:
1.  **Традиционным анализом** (Pine Script индикаторы, скоринговые модели из моего [Market Regime Model](https://github.com/VulpreS/Market-Regime-Signal-Scoring-Model)).
2.  **Предиктивными моделями** (ML-алгоритмы для улучшения качества сигналов).
3.  **Пользовательским интерфейсом** (Telegram-бот для мониторинга и управления).

> **Главная идея:** Использовать ML для пост-фильтрации или верификации сигналов, генерируемых правилами, чтобы снизить количество ложных срабатываний и адаптироваться к меняющимся рыночным режимам.

##  Архитектура проекта

Проект построен по модульному принципу:
Ml-Model/

├── data_fetcher/ # Модуль для загрузки рыночных данных (API бирж, CSV)

├── database/ # Взаимодействие с БД для хранения истории и признаков

├── ml_engine/ # Ядро ML: обучение, предсказания, оценка моделей

├── pine_logic/ # Эмуляция или вызов логики Pine-скриптов из Python

├── telegram_bot/ # Бот для удаленного запуска и получения результатов

├── utils/ # Вспомогательные функции (нормализация, метрики, логирование)

├── config.py # Централизованный файл конфигурации

├── main.py # Точка входа в приложение

├── requirements.txt # Все зависимости

├── .env.example # Пример файла с переменными окружения (токены, пароли)

└── reset_db.py # Скрипт для сброса базы данных




##  Возможности (предполагаемые / roadmap)
Основываясь на текущей структуре, проект может выполнять следующие задачи:

Автоматический сбор данных: Загрузка цен и объемов с криптовалютных или фондовых бирж.

Генерация признаков: Расчет индикаторов (RSI, MACD, EMA) и получение "сырых" сигналов из pine_logic.

Обучение моделей: Классификация рыночных режимов или предсказание направления движения.

Оценка стратегий: Бэктестирование гибридной логики (правила + ML) и сравнение с бенчмарками.

Telegram-интерфейс:

Отправка уведомлений о сильных сигналах.

Запрос статуса моделей (/status).

Запуск переобучения (/retrain).

## Связь с другими проектами
Данный ML-пайплайн является логическим развитием моей многофакторной скоринговой модели.

Скоринговая модель на Pine Script генерирует Composite Score и определяет Market Regime.

ML-Model загружает историю этих оценок, рыночные данные и пытается предсказать итоговую успешность сигнала или скорректировать режим в реальном времени.

## Технологический стек
Язык: Python 3.8+

ML-библиотеки: scikit-learn, (опционально) xgboost, lightgbm, catboost

Работа с данными: pandas, numpy

БД: SQLite / PostgreSQL (определяется в config.py)

API: python-telegram-bot, requests (для биржевых API)

Вспомогательные: python-dotenv
